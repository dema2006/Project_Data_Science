## Идея компании

Мы разрабатываем аналитическую систему для оценки транспортной доступности жилых районов и ЖК в городах России.  
Компания помогает выявлять территории с высокой плотностью населения и слабой транспортной связанностью, чтобы использовать эти данные при городском планировании, развитии общественного транспорта и принятии инфраструктурных решений.

## Разделение обязанностей

- Стаценко Дементий - сбор и парсинг данных о плотности населения в районах городов, а также объединение данных из разных источников для выявления проблемных районов
- Суханов Илья - сбор и парсинг данных о транспортной инфраструктуре, маршрутах и транспортной доступности районов
- Хачатрян Артуш - анализ конкурентов, уже находящихся на рынке
